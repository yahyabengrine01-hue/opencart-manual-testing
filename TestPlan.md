# Test Plan

## Objective

The objective of this testing is to verify that the main functionalities of the OpenCart application work correctly.

## Scope

Modules included in testing:

* User Registration
* Login
* Product Search
* Product Details
* Add to Cart
* Checkout Process

## Testing Types

* Functional Testing
* Negative Testing
* Boundary Testing

## Test Environment

Browser: Google Chrome
Operating System: Windows / Mac

## Test Deliverables

* Test Scenarios
* Test Cases
* Bug Reports
* RTM
* Test Summary Report
